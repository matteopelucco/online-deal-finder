# 🔍 Vinted Scout

Monitoraggio intelligente di Vinted con analisi AI. Trova oggetti da collezione a prezzo vantaggioso, cross-country, con notifiche Telegram istantanee.

## Features

- 🌍 **Cross-country search**: scansiona IT, FR, DE, ES, PL e altri simultaneamente
- 🤖 **AI Analysis**: Claude Haiku valuta ogni listing con score 1-10
- 📱 **Notifiche Telegram**: alert istantanei con foto, prezzo e motivazione AI
- 👥 **Multi-utente**: admin panel per gestire gli accessi
- 📊 **Dashboard**: feed degli alert recenti con statistiche
- ⚡ **Automazione**: cron job GitHub Actions ogni 15 minuti

## Stack

Next.js 14 · TypeScript · Supabase · Vercel · Claude API · Telegram Bot
